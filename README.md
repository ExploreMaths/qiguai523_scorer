# 七怪五二三 计分器（两副牌）网页版

纯前端网页计分程序，使用 HTML + CSS + JavaScript 编写，无需安装，浏览器即可运行。

## 运行方式

### 方式一：直接打开
双击 `index.html`，用浏览器打开即可使用。

### 方式二：本地服务器（推荐）
在文件夹内运行：

```bash
cd D:\qiguai523_scorer
python -m http.server 8080
```

然后在浏览器访问：

```
http://localhost:8080
```

## 功能说明

### 1. 数据自动保存
- 所有数据自动保存到浏览器 `localStorage`。
- 刷新页面或关闭浏览器后重新打开，数据不会丢失。

### 2. 固定组合
- 玩家设置页新增“固定组合”区域。
- 输入组合名称后点击“保存当前为组合”，可把当前玩家列表存为固定组合。
- 点击组合名称一键切换成该组合的玩家。
- 每个组合可“设为默认”；打开页面且无进行中对局时，自动加载默认组合。
- 组合可删除。

### 3. 设置玩家
- 启动后输入玩家姓名，每名玩家一行。
- 点击 🗑 可任意删除某一名玩家。
- 点击“＋ 添加玩家”新增空行。
- 至少需要 2 名玩家，且姓名不能重复。
- 计分过程中可点击顶部“设置玩家”随时返回修改玩家：
  - 新增玩家：过往对局自动补 0 分。
  - 删除玩家：该玩家过往分数会被移除。

### 4. 当前局计分
- 表格显示当前局：
  - 第一行 **原分数**：输入每名玩家的基础扣分/加分。
  - 第二行 **翻倍后分数**：根据本局倍数自动计算并显示。
  - 最右侧 **倍数**：显示当前局总倍数，可撤销或重置。
- 底部显示**本局合计**。

### 5. 炸弹与撤销
- 点击炸弹按钮会按对应倍率累乘到本局倍数：
  - 3 张炸弹：×2
  - 4 张炸弹：×4
  - 5 张炸弹：×8
  - 6 张炸弹：×16
  - 7 张炸弹：×32
- 一局内可连续出多个炸弹，倍数会不断相乘。
- **撤销**：撤销上一次炸弹操作。
- **重置**：倍数回到 ×1。

### 6. 切换局数
- “上一局 / 下一局”查看或编辑历史对局。
- 在最后一局点击“下一局”会自动新增一局。

### 7. 完成查看
- 点击“完成并查看完整表格”，弹出汇总表。
- 每局两行：原分数、翻倍后分数。
- 每局右侧显示该局倍数。
- 最底部显示每名玩家总计和全场总计。
- 正数绿色背景，负数红色背景。
- 序号列和第二列较窄，其余列平均分剩余宽度。

### 8. 重新开始
- 汇总表页面点击“重新开始”可清空所有数据并回到玩家设置。

## 文件说明

- `index.html`：页面结构
- `style.css`：样式
- `app.js`：计分逻辑
